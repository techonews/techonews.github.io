# How to Recover Website from Spam URL Injection

Spam URL injection is a serious SEO security issue where hackers inject thousands of spam pages into a website. These pages usually contain gambling, pharma, or adult keywords that damage search rankings.

## Signs of Spam URL Injection

- Thousands of unknown URLs indexed in Google
- Japanese keyword spam appearing in search results
- Sudden drop in website rankings
- Suspicious pages detected in Google Search Console

## How to Fix Spam URL Injection

### 1. Identify Spam URLs
Use Google search:

site:yourdomain.com

Look for spam pages related to casino, viagra, betting etc.

### 2. Scan Website Files
Scan your website files for malicious scripts or unauthorized PHP files.

### 3. Clean Infected Files
Remove any injected code from your theme files, plugins, or server directories.

### 4. Secure Your Website
- Update CMS and plugins
- Change all passwords
- Enable firewall protection

### 5. Request Google Reindex
After fixing the issue, request reindexing in Google Search Console.
